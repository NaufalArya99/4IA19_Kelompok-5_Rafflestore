# UI/UX Design Documentation

Dokumentasi desain UI/UX untuk proyek Rafflestore.

Desain dibuat menggunakan Figma dan berfungsi sebagai acuan visual.
Implementasi akhir menyesuaikan kemampuan dan framework yang digunakan oleh tim programmer (Laravel).

## Isi Folder
- figma-link.txt : Link desain Figma
- wireframe.png : Wireframe awal
- high-fidelity.png : Desain final (high fidelity)
- ux-process.md : Proses UX
- design-system.md : Sistem desain
- design-deviation.md : Perbedaan desain vs implementasi
